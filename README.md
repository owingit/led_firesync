Analysis of open-loop communication between Photuris frontalis fireflies and 
fixed-frequency driving LED signals. 

usage for paper figures:

python led_analysis.py -i -w -a  --with_stats --do_ffrt --re_norm --do_windowed_period_plot --do_boxplots --save_folder relative/path/to/save/folder

usage for timeseries:

python led_analysis.py -w

requirements:

numpy, scipy, pandas, plotly, matplotlib, seaborn
